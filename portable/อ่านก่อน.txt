VDO Gen Auto Pilot — โฟลเดอร์พกพา (ไม่ต้องติดตั้งอะไรเพิ่ม)
=========================================================

ทุกอย่างอยู่ในโฟลเดอร์นี้แล้ว — Python, adb, scrcpy v4.0, ffmpeg และหน้าเว็บ
ไม่ต้องติดตั้งโปรแกรมใด ๆ บนเครื่อง

วิธีใช้ (ง่ายมาก)
----------------
1. ดับเบิลคลิก  "เปิดโปรแกรม.vbs"
2. รอ 2-3 วินาที เบราว์เซอร์จะเปิดหน้าโปรแกรมให้เอง (http://localhost:3001)
3. ใช้งานผ่านเบราว์เซอร์ได้เลย

ปิดโปรแกรม
----------
ดับเบิลคลิก  "ปิดโปรแกรม.bat"
(หรือปิดเครื่อง/รีสตาร์ทก็ได้)

ตั้งค่าครั้งแรก (ทำครั้งเดียว — ให้คนที่พอรู้เรื่องช่วยตั้ง)
-------------------------------------------------------
1. เปิดโปรแกรม → ไปหน้า "ตั้งค่า/Settings" → ใส่ Google API key
2. เสียบสายมือถือเข้าคอม + เปิด "USB debugging" บนมือถือ → กด Allow
3. ติดตั้งแอป ADBKeyboard บนมือถือ (สำหรับพิมพ์ไทยตอนโพสต์)
4. โหลดส่วนขยาย (extension) เข้า Chrome — สำหรับดูดสินค้า + สั่ง Google Flow

ข้อมูลทั้งหมด (ฐานข้อมูล/ตั้งค่า/คลิป) เก็บในโฟลเดอร์ย่อย "data" ของโฟลเดอร์นี้
ก๊อปทั้งโฟลเดอร์ไปเครื่องอื่น = ยกไปทั้งข้อมูล

ถ้าเบราว์เซอร์ไม่เปิดเอง: เปิดเบราว์เซอร์แล้วพิมพ์  http://localhost:3001
