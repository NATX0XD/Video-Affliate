ดูดสินค้า

เปิดหน้า Shopee Affiliate กดปุ่มจากส่วนเสริม Chrome ดึงชื่อ ราคา รูป และลิงก์เข้าคลังอัตโนมัติ

สร้างคลิปด้วย AI

เลือกสินค้า แล้วให้ Google Flow (Veo) สร้างคลิปสั้นให้ พร้อมแคปชันจาก AI — เลือกแนวและตัวละครได้

รีวิวก่อนโพสต์

ดูคลิปที่ได้ในคลังก่อนปล่อย หรือจะตั้งให้โพสต์อัตโนมัติทันทีเมื่อสร้างเสร็จก็ได้

มือถือโพสต์เอง

ระบบสั่งมือถือ Android จริงผ่านสาย USB เปิดแอป พิมพ์แคปชัน แนบลิงก์ แล้วโพสต์ตามเวลาที่ตั้งไว้

ในโปรแกรมมีอะไร

ออกแบบให้คนไม่เขียนโค้ดใช้ได้จริง

หน้าตั้งค่าครั้งแรกแบบพาทำทีละขั้น

เปิดโปรแกรมครั้งแรกจะมีตัวช่วยพาตั้งค่า 5 ขั้น ไม่ต้องเดาเอง กรอกทีละช่องจนพร้อมใช้

  • ตั้งชื่อร้าน · ใส่คีย์ AI · เชื่อมมือถือ · เลือกแพลตฟอร์ม
  • มีปุ่มทดสอบว่าคีย์และมือถือใช้งานได้จริง
  • คำอธิบายภาษาไทยทุกขั้น ข้ามแล้วมาทำทีหลังได้

ติดตั้งส่วนเสริม Chrome แบบมีคนพาทำ

ส่วนเสริมคือตัวดูดสินค้าและขับ Google Flow — โปรแกรมพาติดตั้งทีละขั้น กดปุ่มเดียวเปิดหน้าตั้งค่าให้ แล้วตรวจจับให้เองว่าติดตั้งสำเร็จ

  • กดปุ่มเปิดหน้าส่วนขยาย + เผยโฟลเดอร์ให้ลากวาง
  • คัดลอกที่อยู่โฟลเดอร์ให้ในคลิกเดียว
  • ตรวจจับอัตโนมัติเมื่อเชื่อมต่อสำเร็จ

คลังคลิปและสินค้ารวมที่เดียว

สินค้าที่ดูดมาและคลิปที่สร้างแล้วอยู่ในคลังเดียว เลือกดู เลือกโพสต์ หรือจัดการเป็นชุดได้

  • ดูสถานะแต่ละคลิป: รอสร้าง / เสร็จ / โพสต์แล้ว
  • เปิด “จอสด” ดูมือถือทำงานจริงตอนโพสต์
  • โหมดอัตโนมัติ: วนโพสต์เอง พักเครื่องเองเมื่อร้อน/แบตต่ำ
ก่อนเริ่ม

ต้องเตรียมอะไรบ้าง

อินเทอร์เน็ตตอนติดตั้ง (โหลดโค้ด + เครื่องมือ) และตอนสร้างคลิป
มือถือ Android + สาย USBเปิด USB debugging และลงแอป ADBKeyboard (พิมพ์ไทย)
บัญชี Googleใช้เข้า Google Flow เพื่อให้ AI สร้างคลิป
Google Chromeสำหรับส่วนเสริม (ดูดสินค้า + ขับ Flow)
เปิดครั้งแรก

เปิดบน Mac ครั้งแรก — ทำครั้งเดียว

แอปยังไม่ได้ซื้อใบรับรอง (unsigned) macOS จะเตือน 1 ครั้ง เป็นเรื่องปกติของเครื่องมือภายใน — ทำ 4 ขั้นนี้ แล้วครั้งต่อไปดับเบิลคลิกเปิดได้เลย

  1. ถ้าเจอกล่องเตือนนี้ (ตามรูป) → กด Done ก่อน (อย่าลบไฟล์)
  2. เปิด เมนู Apple > System Settings > Privacy & Security
  3. เลื่อนลงล่างสุด จะเห็นข้อความ “Install VDO Gen Auto Pilot ถูกบล็อก” → กดปุ่ม Open Anyway
  4. ยืนยันด้วย Touch ID หรือ รหัสเครื่อง → แอปเปิดและเริ่มติดตั้ง (ครั้งต่อไปเปิดได้เลย ไม่ต้องทำซ้ำ)

หมายเหตุ

  • Windows: ถ้าขึ้นจอฟ้า SmartScreen ตอนเปิด ให้กด More info > Run anyway
  • ตัวติดตั้ง ไม่แตะระบบ ลงในโฟลเดอร์ผู้ใช้เอง และถอนออกแล้ว ไม่ลบข้อมูลของคุณ (เก็บแยกต่างหาก)
