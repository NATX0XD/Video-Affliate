# คู่มือเริ่มต้น — VDO Gen Auto Pilot

คู่มือฉบับเดียวจบ: **โปรแกรมนี้ทำอะไร → ติดตั้งยังไง → ใช้งานเบื้องต้นยังไง**
(สำหรับผู้ใช้ทั่วไป ไม่ต้องเขียนโค้ดเป็น)

---

## 1. โปรแกรมนี้ทำอะไร

ระบบสร้างและโพสต์คลิปวิดีโอสั้นแบบ **อัตโนมัติครบวงจร** ตั้งแต่ต้นจนจบ:

```
ดูดสินค้า (Shopee Affiliate)  →  สร้างคลิปวิดีโอด้วย AI  →  โพสต์ลงแพลตฟอร์ม
```

- **ดูดสินค้า** — ดึงชื่อ/ราคา/รูป/ลิงก์ affiliate จาก Shopee ผ่าน Chrome extension
- **สร้างคลิป** — ทำวิดีโอแนวตั้ง 9:16 อัตโนมัติ เลือก engine ได้ 3 แบบ:
  - **Template** — ต่อรูป+ข้อความด้วย ffmpeg (ฟรี)
  - **Avatar** — คนพูดรีวิว (D-ID)
  - **Veo** — วิดีโอ AI จาก Google (เสียเครดิต)
- **โพสต์** — โพสต์ผ่านมือถือจริง (ต่อสาย ADB) ไปได้หลายแพลตฟอร์ม — Shopee Video จูนพร้อมใช้, TikTok/Reels/IG/YouTube มี flow ตั้งต้น

ทั้งหมดทำงานบนเครื่องตัวเอง (local) — ข้อมูล/คลิป/ตั้งค่า เก็บในเครื่อง ไม่ขึ้น cloud

### ส่วนประกอบ 3 ส่วน
| ส่วน | หน้าที่ |
|------|---------|
| **Backend** (`desktop/`) | สมองหลัก — Python server พอร์ต **3001** คุมงานทั้งหมด + เก็บ DB |
| **Extension** (`extension/`) | ปลั๊กอิน Chrome — ดูดสินค้า + สั่ง Google Flow สร้างคลิป |
| **Web UI** (`web/`) | หน้าจอควบคุม — เปิดในเบราว์เซอร์ที่ `localhost:3001` |

## 2. ติดตั้ง (Windows)

> วิธีนี้ลง Python จริงแล้วรันจาก source → **ไม่โดน Antivirus จับ** เหมือนตัว .exe
> ตั้งครั้งเดียว จากนั้นดับเบิลคลิกใช้ได้ทุกวัน

### ขั้นที่ 1 — เอาโค้ดลงเครื่อง
```powershell
git clone https://github.com/Mewwi15/shopee-automation
```
(หรือดาวน์โหลด ZIP แล้วแตกไฟล์ก็ได้)

### ขั้นที่ 2 — ติดตั้ง (ดับเบิลคลิกครั้งเดียว)
**ดับเบิลคลิก `ติดตั้ง.bat`** — แค่นี้จบ ไม่ต้องพิมพ์คำสั่งอะไรเลย
- ถ้าขึ้นหน้าจอฟ้า SmartScreen → กด **More info → Run anyway**
- ตัวติดตั้งจะลงให้อัตโนมัติ: **Python 3.11 · Node.js · adb · scrcpy v4.0 · ffmpeg** (+ ตั้ง PATH), build หน้าเว็บ แล้ว **สร้างทางลัด "VDO Gen Auto Pilot" บนหน้าจอ Desktop** ให้เอง

> ใช้เวลา ~5–10 นาที · ตอนจบจะเช็คให้ว่าเครื่องมือครบไหม · ถ้าติดปัญหาจะขึ้นข้อความบอกวิธีแก้

### ขั้นที่ 3 — ตั้งค่าใช้งานครั้งแรก
1. **เปิดโปรแกรม** — ดับเบิลคลิกทางลัด **"VDO Gen Auto Pilot"** บน Desktop (หรือไฟล์ `เปิดโปรแกรม.vbs` ในโฟลเดอร์) → รอ 2–3 วิ เบราว์เซอร์เด้งเปิด `http://localhost:3001`
2. **หน้าต้อนรับ (ครั้งแรกเท่านั้น)** — ตั้ง **ชื่อร้าน/แบรนด์** → กด **เริ่มใช้งาน**
3. ไปเมนู **ตั้งค่า** → ใส่ **Google API key** (สำหรับให้ AI เขียนแคปชัน/สร้างคลิป)
4. **เสียบมือถือ** + เปิด **USB debugging** บนมือถือ → กด Allow → ลงแอป **ADBKeyboard** (ไว้พิมพ์ไทยตอนโพสต์)
5. **โหลด Chrome extension** — เปิด `chrome://extensions` → เปิด Developer mode → Load unpacked → เลือกโฟลเดอร์ `extension`

เสร็จแล้ว! วันต่อไปแค่ดับเบิลคลิกทางลัด **"VDO Gen Auto Pilot"** บน Desktop พอ

## 2b. ติดตั้ง (Mac)

> ตั้งครั้งเดียว จากนั้นดับเบิลคลิกใช้ได้ทุกวัน (เหมือนฝั่ง Windows)

### ขั้นที่ 1 — เอาโค้ดลงเครื่อง
```bash
git clone https://github.com/Mewwi15/shopee-automation
```
(หรือดาวน์โหลด ZIP แล้วแตกไฟล์)

### ขั้นที่ 2 — ติดตั้ง (ดับเบิลคลิกครั้งเดียว)
**ดับเบิลคลิก `ติดตั้ง.command`** — ไม่ต้องพิมพ์คำสั่งอะไรเลย
- ครั้งแรกถ้า macOS เตือน "ไฟล์จากผู้พัฒนาที่ไม่รู้จัก" → คลิกขวาที่ไฟล์ → **Open** → **Open** อีกครั้ง
- ตัวติดตั้งจะลงให้อัตโนมัติ: **Homebrew · Python · adb · scrcpy · ffmpeg · Python deps** แล้ว **สร้างทางลัด "เปิด VDO Gen Auto Pilot" บนหน้าจอ Desktop** ให้เอง
- ระหว่างลง Homebrew อาจถาม **รหัสเครื่อง (password)** — พิมพ์รหัสเครื่องแล้ว Enter

### ขั้นที่ 3 — ตั้งค่าใช้งานครั้งแรก
ทำเหมือน Windows (หัวข้อ 2 ขั้นที่ 3) — เปิดโปรแกรมจากทางลัด **"เปิด VDO Gen Auto Pilot"** บน Desktop (หรือไฟล์ `เปิดโปรแกรม-mac.command`) แล้วตั้งค่า/ต่อมือถือ/โหลด extension ตามเดิม

## 3. ใช้งานทุกวัน

- **เปิด:** ดับเบิลคลิกทางลัดบน Desktop (Win: `VDO Gen Auto Pilot` · Mac: `เปิด VDO Gen Auto Pilot`) → เบราว์เซอร์เด้งเปิดเอง
- **ปิด:** Win = ดับเบิลคลิก `ปิดโปรแกรม.bat` · Mac = ปิดหน้าต่าง Terminal ที่ค้างอยู่
- **อัปเดตโค้ดใหม่:** Win = ดับเบิลคลิก `update.bat` (ดึงโค้ดใหม่ + build + เปิดใหม่ให้เอง)

> ถ้าเบราว์เซอร์ไม่เด้ง → เปิดเองแล้วพิมพ์ `http://localhost:3001`

### ติดตั้งเป็นแอป (PWA) — เปิดง่ายขึ้น
เปิดหน้าโปรแกรมในเบราว์เซอร์ (Chrome/Edge) → กดไอคอน **ติดตั้ง** (รูปจอมีลูกศรลง) ที่ท้ายช่องพิมพ์ URL → **Install**
จะได้ไอคอนแอปแยกหน้าต่าง (ไม่มีแถบเบราว์เซอร์) เปิดจาก Start Menu / Dock / Launchpad ได้เลย

### หลังอัปเดต — โหลด extension ใหม่ทุกครั้ง
เมื่ออัปเดตโค้ด (เช่นรัน `update.bat`) ถ้าแก้ส่วน extension ด้วย ให้ **reload extension**:
1. เปิด `chrome://extensions`
2. กดปุ่ม **รีโหลด (↻)** ที่การ์ด extension ของโปรแกรม
3. **F5 หน้า Shopee** ที่เปิดค้างไว้ (กัน "context invalidated")

## 4. หน้าจอต่างๆ ทำอะไร

เข้าที่ `http://localhost:3001` เมนูด้านซ้ายจัดเป็น 4 กลุ่ม:

| กลุ่ม | เมนู | ทำอะไร |
|------|------|--------|
| **ภาพรวม** | **ค็อกพิต** | ภาพรวมสถานะระบบ + สรุปงาน + กราฟเผยแพร่ย้อนหลัง |
| **การทำงาน** | **งาน** | คิวงานที่กำลังทำ (สร้างคลิป/โพสต์) |
| | **ผลการโพสต์** | ประวัติการโพสต์ + สถานะ |
| | **คลังคลิป** | คลังสินค้าที่ดูดมา + คลิปที่สร้างแล้ว |
| **ฟาร์มมือถือ** | **ดูแลเครื่อง** | เช็กมือถือที่ต่ออยู่ (ADB) — ต้องเจอเครื่องก่อนถึงโพสต์ได้ |
| | **จอสด** | มิเรอร์หน้าจอมือถือ (ดูตอนโปรแกรมโพสต์) |
| **ระบบ** | **ตั้งค่า** | ตั้งค่าทั้งหมด — API key, engine, แคปชัน, เวลาโพสต์, cooldown ฯลฯ |

> มุมขวาบนมีตัวเลข **มือถือ** (เครื่องที่ต่ออยู่) และ **คิวงาน** (งานค้าง) ให้ดูสถานะเร็วๆ

---

## 5. การทำงานเบื้องต้น (flow)

1. **ดูดสินค้า** — เปิดหน้า Shopee Affiliate → กดปุ่มจาก extension ดูดสินค้าเข้า **คลังคลิป**
2. **สร้างคลิป** — เลือกสินค้า → เลือก engine (Template/Avatar/Veo) → ระบบเข้าคิวสร้างคลิปอัตโนมัติ
3. **รีวิว** — คลิปเสร็จเข้า **คลังคลิป** ให้ดูก่อนโพสต์ (หรือตั้ง auto ให้โพสต์เลย)
4. **โพสต์** — ระบบสั่งมือถือจริงผ่าน ADB โพสต์คลิป + ใส่แคปชัน + ลิงก์ affiliate ตามเวลาที่ตั้งไว้

> ครั้งแรกต้อง **เปิดใช้แพลตฟอร์มปลายทางก่อน** (เช่น Shopee Video) — ค่าเริ่มต้นทุกแพลตฟอร์ม "ปิดอยู่" กดเปิดที่การ์ดแพลตฟอร์มในหน้า **ค็อกพิต** หรือ **ตั้งค่า**

ตั้งเวลา/จำนวนโพสต์ต่อวัน/พักเครื่อง (cooldown เมื่อร้อน/แบตต่ำ) ได้ในหน้า **ตั้งค่า**

| อาการ | วิธีแก้ |
|-------|--------|
| `setup-prereqs.ps1` บอกไม่พบ winget | อัปเดต "App Installer" จาก Microsoft Store ก่อน |
| เปิดแล้วเบราว์เซอร์ไม่เด้ง / ไม่ขึ้นอะไร | ดับเบิลคลิก `_run-source.bat` (โชว์หน้าต่างดำ + error) แล้วส่งรูป error มา |

| หน้า **ดูแลเครื่อง** ไม่เจอมือถือ | เช็ก USB debugging + กด Allow บนมือถือ + ลองสายอื่น |
| โพสต์แล้วพิมพ์ไทยไม่ได้ | ต้องลงแอป **ADBKeyboard** บนมือถือ |
