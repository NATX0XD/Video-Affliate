## 🔧 ข้อ 1 — ติดตั้ง & เปิดโปรแกรมครั้งแรก
**(คู่กับสไลด์ #7)** · เป้าหมาย: ทุกคนเปิด `localhost:3001` ขึ้น + เห็นหน้า **ค็อกพิต**

**พูดว่า:** "ขั้นนี้ทำครั้งเดียวจบ ตั้งเสร็จแล้ววันหลังแค่ดับเบิลคลิกเปิด — ตอนนี้ทำไปพร้อมกันทีละสเต็ป ใครติดยกมือ"

**ทำ (ทีละสเต็ป):**
1. **เอาโค้ดลงเครื่อง** — `git clone https://github.com/Mewwi15/shopee-automation` หรือโหลด ZIP แล้วแตกไฟล์
2. **เปิด PowerShell ในโฟลเดอร์โปรเจกต์** → รัน:
   `powershell -ExecutionPolicy Bypass -File setup-prereqs.ps1`
   - รอ ~5–10 นาที (Python · Node · adb · scrcpy · ffmpeg) → ปล่อยให้รันไป
3. **เปิดโปรแกรม** — ดับเบิลคลิก `เปิดโปรแกรม.vbs` → รอ 2–3 วิ เบราว์เซอร์เด้ง `localhost:3001`
4. **หน้าต้อนรับ (ครั้งแรกเท่านั้น)** — ระบบให้ **ตั้งชื่อร้าน/แบรนด์** → พิมพ์ชื่อ → กด **เริ่มใช้งาน**
5. **ใส่ API key** — ไปเมนู **ตั้งค่า** → ช่อง Google API key → วาง → บันทึก
6. **ต่อมือถือ** — เสียบ USB → เปิด **USB debugging** บนมือถือ → กด **Allow** → เช็กเมนู **ดูแลเครื่อง** ว่าเจอเครื่อง
7. **ลง ADBKeyboard** บนมือถือ (ไว้พิมพ์ไทยตอนโพสต์)
8. **โหลดปลั๊กอิน Chrome** — `chrome://extensions` → เปิด **Developer mode** → **Load unpacked** → เลือกโฟลเดอร์ `extension`

**⚠️ จุดที่มักติด:**
- winget ไม่พบ → อัปเดต "App Installer" จาก Microsoft Store ก่อน แล้วรัน setup ใหม่
- `python`/`adb` ไม่เจอ → ปิด-เปิด PowerShell ใหม่ (PATH เพิ่งเพิ่ม) ถ้ายังไม่ได้รีสตาร์ตเครื่อง
- เบราว์เซอร์ไม่เด้ง → ดับเบิลคลิก `_run-source.bat` ดู error (หน้าต่างดำ) แล้วช่วยดูเป็นราย ๆ
- **ดูแลเครื่อง** ไม่เจอมือถือ → ยังไม่กด Allow / สายเป็นสายชาร์จอย่างเดียว → เปลี่ยนสาย

**เช็กก่อนไปต่อ:** ทุกคนเห็นหน้า **ค็อกพิต** + **ดูแลเครื่อง** เจอมือถือ ✅

## 🔧 ข้อ 2 — ดูดสินค้า → สร้างคลิป
**(คู่กับสไลด์ #12)** · เป้าหมาย: มีสินค้าใน **คลังคลิป** + สั่งคลิป 1 ตัวเข้าคิว

**พูดว่า:** "หัวใจของระบบคือวน 4 ขั้น — ตอนนี้ทำ 2 ขั้นแรก: ดูดของ แล้วสั่งทำคลิป ที่เหลือระบบทำเอง"

**ทำ:**
1. เปิดหน้าสินค้าใน **Shopee Affiliate** (หน้าที่มีลิงก์ affiliate)
2. กดปุ่ม **ดูดสินค้า** จากปลั๊กอิน (ไอคอน/sidepanel) → รอสักครู่
3. กลับมาหน้าเว็บ → เปิดเมนู **คลังคลิป** → เช็กว่าสินค้าเข้ามา (ชื่อ/ราคา/รูป/ลิงก์)
4. เลือกสินค้า → เลือก engine **Template** (ฟรี เห็นผลไว) → กด **สร้างคลิป**
5. เปิดเมนู **งาน** → ดูงานเข้าคิว → รอสถานะเสร็จ
6. กลับ **คลังคลิป** → กดดูคลิปที่ได้ (รีวิว)

**⚠️ จุดที่มักติด:**
- กดดูดแล้วไม่เข้า → รีเฟรชหน้า Shopee (F5) แล้วกดใหม่ (ปลั๊กอินหลุด context)
- ยังไม่ได้ใส่ API key → คลิปสร้างไม่ได้ / แคปชันว่าง → กลับไปเช็ก **ตั้งค่า**
- เลือก Veo แล้วช้า/ถามเครดิต → หน้างานให้ใช้ **Template** ก่อน

**เช็กก่อนไปต่อ:** ทุกคนมีคลิปอย่างน้อย 1 ตัวใน **คลังคลิป** ✅

## 🔧 ข้อ 3 — โพสต์ผ่านมือถือจริง
**(คู่กับสไลด์ #13)** · เป้าหมาย: โพสต์คลิป 1 ตัวสำเร็จ + เห็นมิเรอร์

**พูดว่า:** "ขั้นสุดท้าย ระบบจะสั่งมือถือจริงโพสต์ให้ดู — เปิด **จอสด** จะเห็นมันกดเองเลย"

**ทำ:**
1. เช็กเมนู **ดูแลเครื่อง** — มือถือต้องเชื่อมต่ออยู่ (สถานะเขียว/พร้อม)
2. เปิดใช้แพลตฟอร์ม **Shopee Video** ก่อน (ค่าเริ่มต้นทุกแพลตฟอร์ม "ปิดอยู่" — ดูที่การ์ดแพลตฟอร์มในค็อกพิต/ตั้งค่า)
3. เลือกคลิปที่รีวิวแล้วใน **คลังคลิป** → สั่ง **โพสต์**
4. เปิดเมนู **จอสด** → ดูมือถือทำงานจริง (เปิดแอป พิมพ์แคปชัน แนบลิงก์)
5. เสร็จแล้วเช็กเมนู **ผลการโพสต์** → สถานะ **สำเร็จ**

**⚠️ จุดที่มักติด:**
- พิมพ์ไทยเป็นตัวประหลาด/ว่าง → ยังไม่ได้ตั้ง **ADBKeyboard** เป็นคีย์บอร์ด active บนมือถือ
- มือถือหลับ/ล็อกจอกลางคัน → ตั้งจอไม่ให้ดับ + ปลดล็อกค้างไว้ตอนสาธิต
- โพสต์ค้าง → เช็ก **ดูแลเครื่อง** ว่ายังต่ออยู่ (สายหลุด/หลับ)

**เช็กก่อนจบ:** มีโพสต์ขึ้นสถานะสำเร็จในหน้า **ผลการโพสต์** ✅
