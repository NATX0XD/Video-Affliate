การติดตั้ง

ตั้งครั้งเดียว 3 ขั้น — จากนั้นดับเบิลคลิกใช้ได้ทุกวัน

1
เอาโค้ดลงเครื่องดาวน์โหลดจาก GitHub (git clone หรือโหลด ZIP แล้วแตกไฟล์)
2
รันคำสั่งเดียว setup-prereqs.ps1ลงเครื่องมือให้อัตโนมัติ: Python · Node · adb · scrcpy · ffmpeg (~5–10 นาที)
3
ตั้งค่าใช้งานครั้งแรกเปิดโปรแกรม · ใส่ API key · ต่อมือถือ · โหลดปลั๊กอิน Chrome
วิธีนี้รันจากซอร์สโค้ดจริง → ไม่โดน Antivirus จับ เหมือนไฟล์ .exe
🔧 ลงมือทำ #1

ติดตั้ง & เปิดโปรแกรมครั้งแรก

  • รัน setup-prereqs.ps1 ให้ครบ (ผู้สอนเดินดูทีละคน)
  • ดับเบิลคลิก เปิดโปรแกรม.vbs → เบราว์เซอร์เด้ง localhost:3001
  • ครั้งแรก: หน้าต้อนรับให้ ตั้งชื่อร้าน/แบรนด์ → กดเริ่มใช้งาน
  • ไปเมนู ตั้งค่า → ใส่ Google API key
  • เสียบมือถือ + เปิด USB debugging + ลงแอป ADBKeyboard
  • โหลดปลั๊กอิน: chrome://extensions → Developer mode → Load unpacked → เลือกโฟลเดอร์ extension
▶ ปิดสไลด์ — ทำตาม สคริปต์พาทำ ข้อ 1

เมนูด้านซ้าย จัดเป็น 4 กลุ่ม

ชื่อเมนูจริงในโปรแกรมเป็นภาษาไทยแบบนี้ — จำตามนี้ได้เลย

ค็อกพิต
ภาพรวม สถานะระบบ + สรุปงาน + กราฟย้อนหลัง
งาน
การทำงาน คิวงานที่กำลังทำ (สร้าง/โพสต์)
ผลการโพสต์
การทำงาน ประวัติ + สถานะการโพสต์
คลังคลิป
การทำงาน คลังสินค้าที่ดูด + คลิปที่สร้าง
ดูแลเครื่อง
ฟาร์มมือถือ มือถือที่ต่ออยู่ ต้องเจอก่อนถึงโพสต์ได้
จอสด
ฟาร์มมือถือ มิเรอร์หน้าจอมือถือ ดูตอนโพสต์
ตั้งค่า
ระบบ API key · engine · เวลา · cooldown
มุมขวาบน
ตัวเลข มือถือ = เครื่องที่ต่อ · คิวงาน = งานค้าง
3 เมนูที่ใช้บ่อยสุด: คลังคลิป (ของ) · งาน (คิว) · ดูแลเครื่อง (มือถือ)
วิธีใช้งานจริง

วนซ้ำ 4 ขั้นนี้ทุกวัน

1
ดูดสินค้าเปิดหน้า Shopee Affiliate → กดปุ่มจากปลั๊กอิน → สินค้าเข้า คลังคลิป
2
สร้างคลิปเลือกสินค้า → เลือกสไตล์คลิป → ระบบเข้าคิวทำให้อัตโนมัติ
3
รีวิวคลิปเสร็จเข้า คลังคลิป ดูก่อนโพสต์ (หรือตั้ง auto ให้โพสต์เลย)
4
โพสต์ระบบสั่งมือถือจริงโพสต์ + แคปชัน + ลิงก์ ตามเวลาที่ตั้ง
3 สไตล์คลิป (engine)

เลือกได้ว่าจะให้คลิปหน้าตาแบบไหน

🖼️ Template
ต่อรูปสินค้า + ข้อความเป็นวิดีโอ
ฟรี · เร็ว
🧑 Avatar
คนพูดรีวิวสินค้า (ผ่าน D-ID)
เหมือนคนรีวิว
🎬 Veo
วิดีโอ AI จาก Google คุณภาพสูง
เสียเครดิต
มือใหม่แนะนำเริ่มที่ Template ก่อน — ฟรีและเห็นผลไว จากนั้นค่อยลองแบบอื่น
🔧 ลงมือทำ #2

ดูดสินค้า → สร้างคลิป

  • เปิดหน้าสินค้าใน Shopee Affiliate → กดปุ่มดูดจากปลั๊กอิน
  • กลับมาที่เมนู คลังคลิป → เช็กว่าสินค้าเข้ามาแล้ว
  • เลือกสินค้า → เลือก engine Template → สั่งสร้างคลิป
  • ดูคิวที่เมนู งาน → รอคลิปเสร็จ → รีวิวใน คลังคลิป
▶ ปิดสไลด์ — ทำตาม สคริปต์พาทำ ข้อ 2
🔧 ลงมือทำ #3

โพสต์ผ่านมือถือจริง

  • เช็กเมนู ดูแลเครื่อง ว่าเจอมือถือ (ต้องเขียว/เชื่อมต่อ)
  • เปิดใช้แพลตฟอร์ม Shopee Video ก่อน (ค่าเริ่มต้น "ปิดอยู่")
  • เลือกคลิปที่รีวิวแล้ว → สั่งโพสต์
  • เปิดเมนู จอสด ดูมือถือทำงานจริงตอนโพสต์
  • เช็กผลที่เมนู ผลการโพสต์ → สถานะสำเร็จ
▶ ปิดสไลด์ — ทำตาม สคริปต์พาทำ ข้อ 3
ตั้งค่าที่ควรรู้

ปรับในเมนู ตั้งค่า ให้เข้ากับการใช้งาน

🔑 API key & Engine
ใส่ Google API key + เลือก engine เริ่มต้นที่ใช้บ่อย
⏰ เวลาโพสต์ & จำนวน/วัน
กำหนดช่วงเวลาโพสต์ และโพสต์กี่คลิปต่อวัน
✍️ แคปชัน
รูปแบบข้อความ + ลิงก์ affiliate ที่แนบไปกับโพสต์
🌡️ Cooldown
พักเครื่องอัตโนมัติเมื่อมือถือร้อน/แบตต่ำ กันเครื่องพัง
ตั้งครั้งแรกให้พอใช้ก่อน แล้วค่อยจูนทีหลังเมื่อใช้จริงคล่องขึ้น
