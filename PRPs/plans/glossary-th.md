# อภิธานศัพท์ (Glossary) — ศัพท์ IT → คำง่ายภาษาไทย

> **ที่มา:** งาน P1.2 (ชุด UX พื้นฐาน) · แหล่งจริงของคำอยู่ที่ `web/lib/copy.js`
> **ผู้ใช้ปลายทาง = คนทั่วไป ไม่เขียนโค้ด** — เป้าหมายคือไม่ให้เจอศัพท์เทคนิคดิบ ๆ บนหน้าจอ
>
> **วิธีแก้คำ:** แก้ที่ `web/lib/copy.js` (object `GLOSSARY`) ที่เดียว มีผลทุกหน้าจอทันที
> ไฟล์นี้เป็นแค่ "สรุปให้เจ้าของรีวิว/ปรับคำ" — ถ้าแก้คำในโค้ดแล้ว ควรอัปเดตตารางนี้ให้ตรงกันด้วย

## 1) ศัพท์หลัก (ใช้ในหน้าจอตอนนี้)

| คีย์ (code) | ศัพท์เดิม (IT) | คำที่แสดงให้ผู้ใช้ (TH) | คำอธิบายสั้น (tooltip) |
|---|---|---|---|
| `license_key` | License Key | รหัสเปิดใช้งาน | รหัสที่ได้ตอนซื้อโปรแกรม ใช้ปลดล็อกให้เริ่มใช้งานได้ |
| `machine_id` | Machine ID | รหัสเครื่องคอมนี้ | รหัสประจำเครื่องคอมของคุณ ใช้ผูกกับรหัสเปิดใช้งาน |
| `shop_name` | Shop / Brand | ชื่อร้าน / แบรนด์ | ชื่อร้านของคุณ ใช้แสดงในแคปชันและรายงาน แก้ทีหลังได้ |
| `api_key` | API key | คีย์ AI | รหัสลับสำหรับให้โปรแกรมเรียกใช้ AI แทนคุณ |
| `google_api_key` | Google API Key | คีย์ AI (Gemini) | รหัสลับจาก Google ให้ AI ช่วยคิดข้อความ/สคริปต์คลิป — ขอฟรีที่ aistudio.google.com/apikey |
| `gemini` | Gemini | AI ผู้ช่วยเขียนข้อความ | AI ของ Google ที่ช่วยเขียนคำสั่งและแคปชันให้อัตโนมัติ |
| `veo` | Veo | AI สร้างวิดีโอ (Veo) | AI ที่สร้างคลิปวิดีโอขึ้นมาใหม่จากรูปสินค้า |
| `prompt` | prompt | คำสั่งบอก AI | ข้อความที่เราบอก AI ว่าอยากได้คลิปแบบไหน |
| `render` | render / generating | AI กำลังสร้างวิดีโอ | ช่วงที่ AI กำลังประมวลผลและสร้างคลิปให้ รออีกสักครู่ |
| `caption` | caption | แคปชัน (ข้อความโพสต์) | ข้อความใต้คลิปที่ระบบพิมพ์ให้อัตโนมัติตอนโพสต์ |
| `engine` | engine | วิธีสร้างคลิป | เลือกวิธีทำคลิป เช่น ใช้เทมเพลตฟรี หรือให้ AI สร้างใหม่ |
| `adb` | ADB | การเชื่อมมือถือ | ช่องทางที่คอมสั่งงานมือถือได้ (ต่อผ่านสาย USB หรือ Wi-Fi) |
| `wifi_adb` | Wi-Fi ADB | เชื่อมมือถือผ่าน Wi-Fi | เชื่อมมือถือกับคอมผ่าน Wi-Fi โดยไม่ต้องเสียบสาย — ใส่เลข IP ของมือถือ |
| `adb_keyboard` | ADBKeyboard | คีย์บอร์ดพิมพ์ไทยอัตโนมัติ | ตัวช่วยให้ระบบพิมพ์ภาษาไทยลงมือถือแทนคุณได้ |
| `serial` | serial | รหัสเครื่อง | เลขประจำตัวมือถือแต่ละเครื่อง ใช้แยกว่าเป็นเครื่องไหน |
| `calibrate` | calibrate | จูนพิกัดหน้าจอ | ตั้งตำแหน่งจุดที่ระบบจะแตะบนจอ ให้ตรงกับรุ่นมือถือนั้น |
| `cooldown` | cooldown | พักเครื่อง | ช่วงที่หยุดพักมือถือชั่วคราว เช่น เครื่องร้อนหรือแบตต่ำ |
| `mirror` | mirror / stream | จอสด | ดูหน้าจอมือถือแบบเรียลไทม์บนคอม และคุมเครื่องได้ |
| `desktop` | Desktop app / backend | โปรแกรมหลักในเครื่อง | โปรแกรมที่ต้องเปิดค้างไว้ในคอม เป็นตัวสั่งงานทั้งหมด |
| `extension` | Extension | ส่วนเสริมเบราว์เซอร์ | ตัวเสริมในเบราว์เซอร์ที่ดึงสินค้าและสร้างวิดีโอให้ |
| `server` | server | ตัวประมวลผลในคอม | ส่วนเบื้องหลังในคอมที่คอยรับ-ส่งข้อมูลและทำงานให้ |
| `port` | port | ช่องเชื่อมต่อ | หมายเลขช่องทางที่โปรแกรมใช้คุยกันภายในเครื่อง |
| `endpoint` | endpoint | ที่อยู่เชื่อมต่อระบบ | ที่อยู่ปลายทางที่โปรแกรมส่งข้อมูลไปหา |
| `token` | token | รหัสลับเชื่อมต่อ | รหัสลับที่ใช้ยืนยันตัวตนเวลาเชื่อมต่อระบบ |
| `websocket` | WebSocket | การเชื่อมต่อเรียลไทม์ | ช่องทางรับข้อมูลสด ๆ จากโปรแกรมหลัก (สถานะ/ความคืบหน้า) |
| `sync` | sync | ซิงก์ / อัปเดตให้ตรงกัน | ดึงข้อมูลล่าสุดให้ตรงกันระหว่างมือถือ คอม และหน้าจอนี้ |
| `dry_run` | Dry Run | ทดสอบ (ไม่โพสต์จริง) | ลองรันทั้งขั้นตอนเพื่อตรวจว่าถูกต้องไหม โดยยังไม่โพสต์ออกจริง |
| `platform` | platform | แพลตฟอร์มปลายทาง | ช่องทางที่จะเอาคลิปไปโพสต์ เช่น Shopee, TikTok, Reels |
| `queue` | queue | คิวงาน | รายการงานที่รอทำตามลำดับ |

## 2) ข้อความสำเร็จรูป (MSG) — ใช้ใน toast / เหตุผลปุ่มที่กดไม่ได้

| คีย์ | ข้อความ |
|---|---|
| `offline` | เชื่อมต่อไม่ได้ — ตรวจว่าเปิด "โปรแกรมหลักในเครื่อง" ค้างไว้ แล้วลองใหม่ |
| `apiError` | ทำรายการไม่สำเร็จ — ลองอีกครั้งในอีกสักครู่ |
| `wsLost` | การเชื่อมต่อโปรแกรมหลักหลุด — กำลังลองเชื่อมใหม่อัตโนมัติ |
| `wsBack` | เชื่อมต่อโปรแกรมหลักกลับมาแล้ว |
| `needDesktop` | ยังเชื่อมต่อโปรแกรมหลักในเครื่องไม่ได้ — เปิดโปรแกรมค้างไว้แล้วรอสักครู่ |
| `needDevice` | ยังไม่มีมือถือที่ออนไลน์ — เชื่อมมือถือก่อนจึงจะทำรายการนี้ได้ |
| `saveOk` | บันทึกเรียบร้อยแล้ว |
| `saveFail` | บันทึกไม่สำเร็จ — ลองใหม่อีกครั้ง |

## จุดที่นำไปใช้แล้ว (P1.2)

- **หน้าตั้งค่า** (`app/(app)/settings/page.js`): หัวข้อ/ช่อง "คีย์ AI (Gemini)" + ไอคอน ? อธิบาย, หัวข้อ "แพลตฟอร์มปลายทาง" + อธิบาย "ทดสอบ (ไม่โพสต์จริง)", เด้ง toast ตอนบันทึกสำเร็จ/ไม่สำเร็จ
- **หน้าดูแลเครื่อง** (`app/(app)/devices/page.js`): เช็กลิสต์ความพร้อมมีไอคอน ? อธิบายศัพท์ (การเชื่อมมือถือ / คีย์บอร์ดพิมพ์ไทย / จูนพิกัด ฯลฯ), ไอคอน ? ข้าง Wi-Fi, ปุ่ม "สแกน/เชื่อม" ปิดพร้อมบอกเหตุผลเมื่อยังไม่เชื่อมต่อโปรแกรมหลัก
- **หน้าฟาร์มมือถือ** (`app/(app)/mirror/page.js`): ปุ่มสแกน/เชื่อม ใช้ `GatedButton` (ปิด + บอกเหตุผล), ไอคอน ? ข้าง Wi-Fi
- **หน้าต้อนรับ/ตั้งค่าครั้งแรก** (`components/Onboarding.js`): ไอคอน ? อธิบายช่องชื่อร้าน

## หมายเหตุการปรับคำ

- คำแนะนำ: ให้สั้น เข้าใจใน 1 บรรทัด หลีกเลี่ยงภาษาอังกฤษถ้ามีคำไทยที่สื่อได้
- ศัพท์ที่ผู้ใช้ "ต้องพิมพ์/ค้นเจอจริง" (เช่น `AIza…` ของคีย์, ชื่อ `ADBKeyboard` ตอนติดตั้ง) คงคำอังกฤษไว้ในวงเล็บได้ เพื่อให้หาต่อได้
- ถ้าเพิ่มศัพท์ใหม่: เพิ่มใน `GLOSSARY` ของ `web/lib/copy.js` แล้วเติมแถวในตารางนี้
